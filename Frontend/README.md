# Finances Pro Frontend

Frontend Vue 3 para Finances Pro.

## Inicio Rápido

```bash
npm install
npm run dev
```

URL local:

```text
http://localhost:5173
```

## Comandos

```bash
npm run dev
npm run build
npm run preview
npm run lint
npm run format
```
